# 🗳️ Digital Voting System

A simple web-based Digital Voting System built using HTML, CSS, and JavaScript. This project provides an interactive interface where users can cast votes for different political parties and view the vote count instantly.

## 📸 Project Preview

The application includes:

- Modern voting interface
- Multiple party selection using radio buttons
- Vote submission functionality
- New voting session option
- Vote counting and result display
- Responsive and user-friendly design

## 🚀 Features

✅ Cast votes digitally

✅ Prevent multiple party selections

✅ Count votes automatically

✅ Simple and lightweight implementation

✅ Easy to understand for beginners

✅ Pure frontend implementation using HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

⚙️ How to Run
Clone the repository:

git clone https://github.com/your-username/digital-voting.git
Open the project folder.

Run index.html in your browser.

OR

Use VS Code Live Server:

Right Click → Open with Live Server
🎯 Working
Select a political party.

Click Save to submit your vote.

Click New Vote to allow another voter.

Click Count Vote to view the voting results.
